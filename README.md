# Acernity Portfolio

A freelance portfolio template guided by Manu Arora founder of Acernity UI. This portfolio template uses react motion to create micri interactions and tasteful design 😋

## Project Description

This is a premium freelance portfolio template built with modern web technologies. It focuses on high-quality micro-interactions, fluid animations, and a sleek, professional aesthetic designed to showcase your work in the best possible light.

## Tech Stack

- **Framework**: [Next.js](https://nextjs.org)
- **Styling**: [Tailwind CSS](https://tailwindcss.com)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **UI Components**: [Aceternity UI](https://ui.aceternity.com/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
